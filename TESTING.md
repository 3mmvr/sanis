# Testing Instructions

## Quick Start
1. Visit the deployed app on Netlify
2. Click **"Continue as Guest"** on the login page
3. Complete the 5-stage onboarding flow
4. Create a pet profile (add name, breed, weight, age)
5. Test the scanner by uploading/capturing a pet meal image
6. View meal analysis, nutrition breakdown, and AI insights
7. Check the Analytics page for streaks and weekly stats
8. Test hydration tracking and settings

**Note**: Guest mode stores data locally. Use for testing purposes only.
